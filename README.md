# 🔔 청소년 맞춤형 자원봉사 알리미 웹앱

1365 자원봉사포털 기반으로 순천시, 광양시, 진주시 지역의 청소년 가능 봉사활동(성인 동반, 헌혈, 비대면 포함)을 필터링하여 보여주는 Streamlit 웹 애플리케이션 저장소입니다.

## 🚀 로컬 실행 방법

1. 터미널에서 필요한 패키지를 설치합니다:
   `pip install -r requirements.txt`
2. 앱을 실행합니다:
   `streamlit run app.py`

## 🌐 GitHub을 통한 Streamlit Cloud 배포 방법

1. GitHub에 새로운 저장소(Repository)를 생성합니다.
2. 이 폴더 안에 있는 파일들(`app.py`, `requirements.txt`, `README.md`, `.gitignore`)을 GitHub 저장소에 업로드(Push)합니다.
3. Streamlit Community Cloud (https://share.streamlit.io/) 에 접속하여 GitHub 계정으로 로그인합니다.
4. **New app** 버튼을 누르고 방금 만든 GitHub 저장소와 `app.py` 경로를 지정한 뒤 **Deploy**를 클릭합니다.
